### 실습환경

- ubuntu 18.04 LTS
- opencv 3.4.0
- cuda 10.0
- NVIDIA Xavier
- Logitec HD Usb camera (1080p)

- 고정된 카메라로부터 마커를 통해 검출된 좌표기반으로 픽셀좌표 <-> 실제좌표를 비교하여 물체의 거리를 예측함
